# Python_List-Tuples-Questions
This repository contains a set of 10 Python practice problems based on Lists and Tuples, designed to strengthen your understanding of data collection types in Python. 
These problems focus on list/tuple creation, iteration, searching, sorting, type conversion, and performing operations without relying on built-in shortcuts.

All questions are part of a larger set of 50 Python beginner-friendly exercises, and this section includes Questions 19 to 28.
